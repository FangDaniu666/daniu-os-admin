# daniu-os-admin

daniu-os的后台实现，基于 [vue-naive-admin](https://github.com/zclzone/vue-naive-admin) 后台管理模板。

## 运行环境
- 建议使用 Node.js 版本 14 或更高版本。

## 快速开始

按照以下步骤快速启动项目：

```sh
npm install
npm run dev
```

如果你的 npm 和 vue-cli 版本正确，以上命令应该可以启动程序。项目使用的具体版本可参考上文和 `package.json` 文件。
